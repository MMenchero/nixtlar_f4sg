# Foundation Time Series Models: An Application for Retail Forecasting 

This repository contains the materials for the *Forecasting 4 Social Good* workshop on `nixtlar`.  

`nixtlar` is an R package that provides an interface to *TimeGPT*, a foundation model for time series forecasting developed by Nixtla. It enables fast and accurate forecasting with minimal configuration, making it particularly useful for retail demand forecasting and other applications.  

## Included Scripts  

- `program.R` – Main workshop script featuring an example using the VN1 Competition dataset.

- `functions.R` – Utility functions used in `program.R`.  

- `autotimegpt.R` – Automatic fine-tune selection using TimeGPT. 

- `visualization.R` – Interactive forecast visualization using the `trelliscope` R package.
