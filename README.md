# Foundation Time Series Models: An Application for Retail Forecasting 
This repository contains materials for the Forecasting 4 Social Good workshop on `nixtlar`.

[nixtlar](https://nixtla.github.io/nixtlar/) is an R package that provides an interface to TimeGPT, a foundation model for time series forecasting developed by Nixtla. It enables fast, accurate forecasting with minimal configuration, making it particularly useful for applications in retail demand forecasting. 

## Included scripts: 
- Main code for the VN1 Competition example 
- Helper functions
- AutoTimeGPT version
- Visualiation with `trelliscope`
